# Project: FIFA World Cup tracker
#ai/project

30/04/2026

This year is the FIFA World Cup. I want to build a fun and interactive web-based World Cup fixtures tracker and news aggregator that I can enjoy with my family. For the moment, it would just be a Vercel website, and I may buy a URL for it at some point. It needs to be something that we can use to see when the games are being played, where we can watch them on UK TV, and any news about players, etc., and enjoy it during the tournament.

My understanding is that there is a FIFA RSS feed, but they don't have an API. That is something you can check. There are other sources of information out there, such as Opta and Who Scored, and also these sources.

* **OPTA**: A comprehensive but commercial database used by services like Whoscored.com. 
* **Football-Data.org**: A free, stable API in active development. 
* **RSS Parsing**: Projects like the **FIFA-News-API** on GitHub demonstrate how to ingest and parse FIFA's RSS feeds for news updates.

Plan out how you will build this in a way that is efficient and free. Visually, it should have world cup themes and have a good level of interactivity, something that a child would enjoy.
